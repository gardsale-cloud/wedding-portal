# Wedding Portal

Небольшая заготовка структуры проекта для свадебного портала.

Структура:
- public/: статичные файлы (index.html, favicon)
- src/styles/: стили
- src/pages/: страницы
- src/js/: скрипты
- src/assets/images/: изображения

Инструкции: откройте public/index.html в браузере или используйте простой статический сервер.
